웹을 짜기 시작하는 가장 가벼운 첫 화면

WEAVE는 더 빠르고 정돈된 퍼블리싱을 위해 웹을 직조하듯 구성하는 도구입니다. 이 시작 페이지는 정적 프로젝트에 바로 옮길 수 있는 기본 자산 구조와, 프로젝트 커스터마이징을 위한 `prj.css`, `prj.js` 진입점까지 함께 준비해 둔 최소 베이스입니다.

Base CSS `wv.css`, `common.css`, `admin.css`, `gov.css`와 함께 `prj.css`를 바로 덧입힐 수 있습니다.
Base JS `core.js` 위에 프로젝트 전용 스크립트인 `prj.js`를 별도로 분리해 시작할 수 있습니다.
Ready Layout 첫 화면부터 브랜드와 자산 구조를 확인한 뒤, Docs 가이드 흐름대로 검증을 이어갈 수 있습니다.

먼저 기본 자산을 연결하고, 이후 문서에 있는 CSS/JS 예시를 같은 구조에 붙여 검증하는 흐름을 권장합니다.

Project Entry 기본 CSS와 런타임 위에 `prj.css`, `prj.js`를 얹는 구조로 시작합니다.
Folder Direction
  • `public/css` : 기본 스타일 + 프로젝트 오버라이드
  • `public/js` : 런타임 + 프로젝트 전용 스크립트
  • `public/plugins`, `public/images/*` : 이후 확장 자산 위치
Docs Ready 문서에 적힌 마크업과 플러그인 등록 예시를 그대로 옮겨 검증하기 좋은 스타트 포인트입니다.
