# WEAVE

WEAVE is a publishing-oriented frontend library built around CSS utilities, a plugin runtime, and a docs app for demos and validation.

## KOR

WEAVE는 퍼블리싱 작업 흐름에 맞춘 CSS 유틸리티 + JavaScript 플러그인 라이브러리입니다.

- 사용 가능 기능: `wv.css`, `common.css`, `admin.css`, `gov.css`, `createWeave()`, `copy`, `fileDrop`, `linkButton`, `targetButton`, `tabs`, `accordion`, `modal`, `hideToday`, `scrollTo`, `anim(fade/parallax/cascade/marquee/ticker/count)`, `validation`
- 문서 페이지: `Home`, `Docs(CSS/JS)`, `Anim`, `Validation`, `Form`, `Plate`, `Download`, `Tools`
- 각 페이지에서 가이드, 데모, 토큰, 다운로드 흐름을 바로 확인할 수 있습니다.

## ENG

WEAVE is a frontend library for publishing workflows, combining CSS utilities with a plugin-based JavaScript runtime.

- Available features: `wv.css`, `common.css`, `admin.css`, `gov.css`, `createWeave()`, `copy`, `fileDrop`, `linkButton`, `targetButton`, `tabs`, `accordion`, `modal`, `hideToday`, `scrollTo`, `anim(fade/parallax/cascade/marquee/ticker/count)`, `validation`
- Docs pages: `Home`, `Docs(CSS/JS)`, `Anim`, `Validation`, `Form`, `Plate`, `Download`, `Tools`
- The docs app provides guides, demos, tokens, and download flows in one place.
